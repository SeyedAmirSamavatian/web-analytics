# استفاده از IP محلی (فقط برای شبکه محلی)

## محدودیت‌ها:
- فقط در همان شبکه محلی کار می‌کند (همان WiFi)
- نیاز به تنظیمات فایروال دارد
- امنیت کمتر

## مراحل:

1. IP محلی خود را پیدا کنید:
   ```bash
   # Windows
   ipconfig
   
   # Linux/Mac
   ifconfig
   ```
   
   IP شما چیزی شبیه `192.168.1.100` خواهد بود.

2. Backend را روی IP محلی اجرا کنید:
   ```bash
   # در فایل .env یا هنگام اجرا
   HOST=0.0.0.0 PORT=3000 npm run dev
   ```

3. در سایت خود از IP استفاده کنید:
   ```html
   <script 
     src="http://192.168.1.100:3000/tracker.js" 
     data-key="YOUR_TRACKING_KEY"
     data-api="http://192.168.1.100:3000">
   </script>
   ```

**نکات:**
- فقط دستگاه‌هایی که در همان شبکه WiFi هستند می‌توانند دسترسی داشته باشند
- فایروال Windows ممکن است نیاز به تنظیم داشته باشد
